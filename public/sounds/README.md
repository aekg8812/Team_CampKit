# 効果音・BGM ファイルをここに置く

以下のファイル名で mp3 を配置してください（フリー素材でOK）。
ファイルが無くても再生がスキップされるだけで、アプリは正常に動作します。

## 効果音

- `gavel.mp3` … 木槌「ガンッ」（ガチャ停止時）
- `roulette.mp3` … ルーレット回転音（ループ再生）
- `alarm.mp3` … 重刑(HIGH)確定時の警報
- `reveal.mp3` … 予言のタイピング音
- `clear.mp3` … 執行完了
- `fail.mp3` … 執行失敗

## BGM（今回追加）

- `bgm_question.mp3` … 質問10問画面のBGM（ループ）— 緊張感のある BGM を推奨
- `bgm_gacha.mp3` … ガチャ演出画面のBGM（ループ）— ドラマチック・サスペンス系を推奨
