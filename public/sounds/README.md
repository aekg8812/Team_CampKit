# 効果音・BGM ファイルをここに置く

`public/sounds/` に以下のファイル名で `.mp3` を置くだけで自動的に再生されます。
ファイルが無くても再生はスキップされるだけで、アプリは正常に動作します。

---

## 置くべきファイル一覧

| ファイル名 | 使用場面 | 種類 | 推奨イメージ |
|---|---|---|---|
| `gavel.mp3` | ガチャ停止時 | 効果音 | 木槌「ガンッ」 |
| `roulette.mp3` | ガチャ回転中（ループ） | 効果音 | カチカチ / スロット回転 |
| `alarm.mp3` | 激重課題が出たとき | 効果音 | 警報・サイレン |
| `reveal.mp3` | 診断テキストのタイピング中 | 効果音 | タイプライター音 |
| `clear.mp3` | 課題達成（成功）時 | 効果音 | ファンファーレ・クリア音 |
| `fail.mp3` | 課題失敗時 | 効果音 | ブザー・失敗音 |
| `bgm_question.mp3` | 質問10問画面（ループ） | BGM | 緊張感のある BGM |
| `bgm_gacha.mp3` | ガチャ演出画面（ループ） | BGM | ドラマチック・サスペンス系 |
| `omikuji_shake.mp3` | おみくじ箱を振ったとき | 効果音 | カラカラ / シャラシャラ |
| `omikuji_result.mp3` | おみくじ結果発表時 | 効果音 | ジャーン / ファンファーレ |

---

## 手順

1. 下の素材サイトから mp3 をダウンロードする
2. ファイル名を上の表の通りにリネームする
3. このフォルダ（`public/sounds/`）に入れる
4. `npm run dev` でブラウザを開くと即再生される（再起動不要）

新しいサウンドを追加したい場合は `src/lib/sound.js` の `SOUND_SRC` にキーとパスを追加し、
使いたい場所で `play("キー名")` を呼ぶだけです。
